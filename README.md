# Smartdustbin_Arduino
Arduino-based smart dustbin with ultrasonic, Bluetooth and motion sensor

#Features
-Ultrasonic sensor to detect dustbin fill level
- Bluetooth-based alert when dustbin reaches threshold
- Motion sensor for automatic lid operation

## Hardware Used
- Arduino UNO
- Ultrasonic Sensor (HC-SR04)
- PIR Motion Sensor
- Bluetooth Module (HC-05 / HC-06)

## Software
- Arduino IDE

## How It Works
The ultrasonic sensor measures the distance to detect waste level.
When the threshold is reached, a Bluetooth alert is triggered.
The motion sensor enables automatic lid operation.
